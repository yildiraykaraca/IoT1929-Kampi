UZAKTAN SERVO MOTOR

Problem Tanımı:

Neler olmalı?

· Nodemcu sensörü

· Servo motor

· Blynk uygulması

· Devrenin kodlarını, devre şemasını ve devrenin çalışır halini bizimle paylaşmalısın

· En pratik çözüm olmalıdır
